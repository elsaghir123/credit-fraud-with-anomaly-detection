# 💳 Credit Fraud Detection with Anomaly Detection

## 📊 Dataset

The dataset used in this project is the **Credit Card Fraud Detection Dataset**, available on Kaggle:

🔗 https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

---

## 📌 Context

Credit card fraud detection is critical for financial institutions to prevent customers from being charged for transactions they did not authorize.

The goal of this project is to detect fraudulent transactions using anomaly detection and machine learning techniques.

---

## 📁 Dataset Description

The dataset contains credit card transactions made in **September 2013 by European cardholders**.

- Total transactions: **284,807**
- Fraudulent transactions: **492**
- Fraud percentage: **0.172%**

⚠️ The dataset is **highly imbalanced**, making it a challenging classification problem.
